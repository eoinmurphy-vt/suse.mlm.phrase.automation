== 인증 제공업체란 무엇을 의미하나요?

인증 제공업체는 Turtles 팀의 CI 워크플로에서 **적극적으로 테스트하여** Rancher용 CAPI 확장 프로그램과 제대로 작동하는지 검증하는 CAPI 제공업체입니다. 그러나 대부분의 CAPI 프로바이더는 쿠버네티스 오픈 커뮤니티에서 유지 관리하는 업스트림 프로젝트이므로, 새로운 프로바이더 버전이 출시될 때 문제가 발생하지 않는다고 **보장할 수는 없습니다**.
