.가져온 인증서 제거
[NOTE]
====
이전 버전에서는 데이터베이스 컨테이너에 다음 주체 대체 이름(SAN)이 포함된 SSL 인증서가 필요했습니다:

* `+db+`
* `+reportdb+`

더 이상 필요하지 않습니다.
데이터베이스 컨테이너는 외부에 노출된 완전한 도메인 이름만 필요합니다.
`+db+` 및 `+reportdb+` SAN이 포함된 이전 인증서를 계속 사용할 수 있습니다.

같은 인증서를 메인 컨테이너와 데이터베이스 컨테이너 모두에 사용할 수 있습니다.

새 인증서를 업그레이드 명령에 전달하려면 `+--ssl-db-ca-root+`, `+--ssl-db-cert+` 및 `+--ssl-db-key+` 매개변수를 사용하십시오.
====
