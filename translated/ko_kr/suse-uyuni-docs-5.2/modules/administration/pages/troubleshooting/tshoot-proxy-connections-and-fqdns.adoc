= 프록시 연결 및 FQDN 문제 해결
:revdate: 2022-10-18
:page-revdate: {revdate}

때때로 프록시를 통해 연결된 클라이언트가 {webui}에 나타나지만, 프록시를 통해 연결되어 있다는 표시가 나타나지 않습니다.
이것은 완전한 도메인 이름(FQDN)을 사용하여 연결하지 않거나 프록시가 {productname}에 알려지지 않은 경우 발생할 수 있습니다.

이러한 동작을 교정하려면 다음과 같이 프록시의 클라이언트 구성 파일에서 추가 FQDN을 grain으로 지정하십시오.

----
grains:
  susemanager:
    custom_fqdns:
      - name.one
      - name.two
----
