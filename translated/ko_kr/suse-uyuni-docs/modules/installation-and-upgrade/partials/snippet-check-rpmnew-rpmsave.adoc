. 컨테이너 내부에서 [filename]`+*.rpmnew+` 및 [filename]`+*.rpmsave+` 파일을 확인하십시오:

+

[source,shell]
----
mgrctl exec 'find /etc/ -name "*.rpmnew" -o -name "*.rpmsave"'
----

+

[NOTE]
====
패키지 설치 후 변경된 기본 구성 파일에 대한 변경 사항이 업그레이드에 포함된 경우 파일을 덮어쓰는 대신 이러한 파일 유형 중 하나가 생성됩니다:

* [filename]`+*.rpmnew+`: 새로운 기본 구성 파일을 포함하고, 변경된 구성 파일은 그대로 유지합니다.
* [filename]`+*.rpmsave+`: 새로운 기본 구성 파일로 대체된 변경된 구성 파일의 복사본입니다.
====

+

. [filename]`+*.rpmnew+` 또는 [filename]`+*.rpmsave+` 파일이 발견되면 내용을 검사하고 원하는 변경 사항을 병합하십시오.
