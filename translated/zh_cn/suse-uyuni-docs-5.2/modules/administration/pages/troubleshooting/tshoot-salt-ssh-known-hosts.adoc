= 排查 Salt SSH 已知主机问题
:revdate: 2026-04-15
:page-revdate: {revdate}

如果机器或虚拟机重装了新的 SSH 密钥，但仍使用相同的 IP 地址进行接入，Salt 可能会提示“中间人攻击”。
这是因为 Salt 的 `+.ssh/known_hosts+` 文件中仍包含旧的 SSH 指纹。

要从该列表中去除主机，请运行以下命令：

[source,shell]
----
mgrctl ssh knownhost remove <hostname>
----

如果要去除的主机不使用默认的 SSH 端口 `+22+`，您可以手动指定端口：

[source,shell]
----
mgrctl ssh knownhost remove <hostname> <port>
----
