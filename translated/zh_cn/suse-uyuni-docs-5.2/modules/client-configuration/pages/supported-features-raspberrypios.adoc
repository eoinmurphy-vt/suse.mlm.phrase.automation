[[supported-features-raspberrypios]]
= 支持的 {raspberrypios} 功能
:revdate: 2025-06-30
:page-revdate: {revdate}

下表列出了 {raspberrypios} 客户端上各种功能的可用性。

[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。{raspberrypios} 由 {raspberrypios} 社区支持。
====

此表中图标的含意如下：

* {check} 该功能可用
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
