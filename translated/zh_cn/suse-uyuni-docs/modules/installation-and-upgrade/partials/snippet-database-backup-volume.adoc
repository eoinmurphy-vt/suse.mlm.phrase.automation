= 数据库备份卷
:revdate: 2025-07-29
:page-revdate: {revdate}

通过 [command]``+mgradm migration+``+ 或 [command]+``+mgradm upgrade+`` 执行服务器迁移或升级时，可以创建一个包含数据库备份的卷。

当 {postgresql} 数据库版本增加时，旧数据库必须在运行升级之前存储在一个单独的位置。
为此，[command]``+mgradm+``+ 动态创建卷 [path]+``+var-pgsql-backup+``。
当迁移或升级完成后，用户验证新系统正常工作后，可以安全地去除该卷。
