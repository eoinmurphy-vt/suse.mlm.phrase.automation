[[proxy-setup-containers-transfer-config]]
== 传输 {productname} 代理配置


[command]``+spacecmd+`` 命令和通过 {webui} 方式生成的配置归档文件均可生成配置归档。
该归档需要在容器主机上可供使用。
将生成的存档传输到容器主机。



[[proxy-setup-containers-transfer-start]]
== 启动 {productname} 代理容器

可以使用 [literal]``+mgrpxy+`` 命令启动容器。

[[proc-setup-containers-setup-start]]
.过程：升级启动 {productname} 代理容器

. 运行以下命令：
+
----
mgrpxy start uyuni-proxy-pod
----
+
. 通过调用以下命令检查是否所有容器都已按预期启动
+
----
podman ps
----

应该有五个 {productname} 代理容器，并且它们应该是 [literal]``+proxy-pod+`` 容器 Pod 的一部分。

* proxy-salt-broker
* proxy-httpd
* proxy-tftpd
* proxy-squid
* proxy-ssh
