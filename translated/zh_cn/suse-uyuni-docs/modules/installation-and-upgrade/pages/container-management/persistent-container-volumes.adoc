[[persistant-volume-list]]
= 永久性卷
:description: 学习如何访问和管理 SUSE Multi-Linux Manager 的持久容器卷
:revdate: 2025-07-29
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

在容器内进行的修改不会被保留。
在持久卷之外所做的任何更改将被丢弃。
以下是 {productname} {productnumber} 的持久卷列表。

要自定义默认卷位置，请确保在首次启动 Pod 之前使用 [command]``+podman volume create+`` 命令创建必要的卷。

[NOTE]
====
请确保此表格与 Helm 图表和 systemctl 服务定义中所述的卷映射完全一致。
====
