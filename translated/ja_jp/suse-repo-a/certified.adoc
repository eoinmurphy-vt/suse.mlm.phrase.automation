== 認定プロバイダーとはどういう意味ですか？

認定プロバイダとは、TurtlesチームのCIワークフローによって**積極的にテストさ**れ、Rancher用CAPIエクステンションで正しく動作することが確認されたCAPIプロバイダのことです。しかし、ほとんどのCAPIプロバイダーは、Kubernetesオープンコミュニティによって保守されているアップストリームプロジェクトであるため、新しいプロバイダーのバージョンがリリースされたときに問題が発生しないことを**保証することはできません**。

== 認定プロバイダー向けクラスタ・クラス・サポート

以下は、各認証プロバイダーとクラスタ・トポロジー機能のサポート・マトリックスである：

[tabs]
======
CAPZ::
+
--
- `+ClusterClass+` の**完全サポート**：マネージドクラスター（AKS）とアンマネージドクラスター（仮想マシン）の両方をトポロジー経由でプロビジョニングできる。
--

CAPA::
+
--
- `+ClusterClass+` の**完全サポート**：マネージド（EKS）クラスタとアンマネージド（EC2ベース）クラスタの両方を、トポロジー経由でプロビジョニングできる。
--

CAPG::
+
--
- `+ClusterClass+` の**完全サポート**：管理クラスタ（GKE）と非管理クラスタ（仮想マシン）の両方を、トポロジー経由でプロビジョニングできます。
