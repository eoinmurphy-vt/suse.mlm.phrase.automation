[[proxy-setup-containers-transfer-config]]
== プロキシ{productname}設定を転送します


`+spacecmd+`コマンドおよび{webui}の方法の両方で、設定アーカイブが作成されます。
このアーカイブはコンテナホストで利用可能にする必要があります。
この生成されたアーカイブをコンテナホストに転送します。



[[proxy-setup-containers-transfer-start]]
== プロキシ{productname}コンテナを開始します

`+mgrpxy+`コマンドを使用してコンテナを起動できます。

[[proc-setup-containers-setup-start]]
.プロシージャ:プロキシ{productname}コンテナを開始します

. コマンドを実行します。
+
----
mgrpxy start uyuni-proxy-pod
----
+
. 次のコマンドを呼び出して、すべてのコンテナが期待どおりに起動したかどうかを確認します。
+
----
podman ps
----

5つの{productname}プロキシコンテナが存在し、`+proxy-pod+`コンテナポッドの一部である必要があります。

* proxy-salt-broker
* proxy-httpd
* proxy-tftpd
* proxy-squid
* proxy-ssh
