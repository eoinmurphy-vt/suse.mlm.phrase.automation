[[opensuse-registration-overview]]
= openSUSEクライアントの登録
:revdate: 2024-02-22
:page-revdate: {revdate}

{productname}サーバに{opensuse}および{leapmicro}クライアントを登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバーと日時を正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。


[IMPORTANT]
====
{productname}サーバを自分自身に登録しないでください。
{productname}サーバは、個別に管理するか、別の{productname}サーバを使用して管理する必要があります。
複数のサーバを使用することに関する詳細については、xref:specialized-guides:large-deployments/hub-multi-server.adoc[]を参照してください。
====
