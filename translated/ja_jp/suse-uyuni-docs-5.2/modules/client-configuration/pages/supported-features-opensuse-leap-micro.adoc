= {leapmicro} クライアント機能
:revdate: 2024-02-01
:page-revdate: {revdate}


[IMPORTANT]
====
{leapmicro} は {suse} コミュニティによってサポートされています。
====


この表のアイコンの意味は次のとおりです。

* {check}: 機能は使用可能です
* {cross}: 機能は使用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
