[[almalinux-registration-overview]]
= {almalinux} クライアントの登録
:revdate: 2024-01-31
:page-revdate: {revdate}

あなたの{productname}サーバに{almalinux}クライアントを登録できます。
その方法および詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントの日時が{productname}サーバと正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。

* アクティベーションキーの作成に関する詳細については、xref:client-configuration:activation-keys.adoc[]を参照してください。
* {almalinux}を{sll}に移行することに関する詳細については、xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]を参照してください。
