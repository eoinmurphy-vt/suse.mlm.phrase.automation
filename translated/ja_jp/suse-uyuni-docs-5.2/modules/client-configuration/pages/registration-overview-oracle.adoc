[[oracle-registration-overview]]
= Oracleクライアントの登録
:revdate: 2024-01-31
:page-revdate: {revdate}

{oracle}クライアントをお使いの{productname}サーバに登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバーと日時を正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。

* アクティベーションキーの作成に関する詳細については、xref:client-configuration:activation-keys.adoc[]を参照してください。
* {oracle}を{sll}に移行することに関する詳細については、xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]を参照してください。
