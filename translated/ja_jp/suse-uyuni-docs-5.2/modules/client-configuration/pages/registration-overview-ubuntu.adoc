[[ubuntu-registration-overview]]
= Ubuntuクライアントの登録
:revdate: 2020-08-27
:page-revdate: {revdate}

{productname}サーバに{ubuntu}クライアントを登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバーと日時を正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
