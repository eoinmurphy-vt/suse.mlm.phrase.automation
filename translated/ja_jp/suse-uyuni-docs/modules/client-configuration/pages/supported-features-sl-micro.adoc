[[supported-features-sl-micro]]
= サポートされている{sl-micro}クライアント機能
:description: SL Micro Clientのサポートされている機能を探ります。これには、オペレーティングシステムパッケージ、登録、およびインストールオプションが含まれます。
:revdate: 2024-07-18
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


[IMPORTANT]
====
クライアントで実行しているオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。{sl-micro}は{suse}でサポートされています。
====

[IMPORTANT]
====
{sl-micro}は、現時点では通常のミニオン（[systemitem]``+default+``+接続方法）としてのみサポートされています。私たちは、Salt SSHクライアント（[systemitem]+``+salt-ssh+``接続方法）として管理する作業も進めています。
====

この表のアイコンの意味は次のとおりです。

* {check}: 機能は使用可能です
* {cross}: 機能は使用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
