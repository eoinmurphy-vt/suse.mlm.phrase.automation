[NOTE]
====
セキュリティのために強化された{productname}サーバーホストは、[path]``+/tmp+``フォルダーからのファイルの実行を制限する場合があります。
そのような場合、回避策として、[command]+``+mgradm+``を実行する前に、[literal]``+TMPDIR+``+環境変数を別の既存のパスにエクスポートしてください。

例:

[source,shell]
----
export TMPDIR=/path/to/other/tmp
----

{productname}の更新では、この回避策が不要になるようにツールが変更されます。
====
