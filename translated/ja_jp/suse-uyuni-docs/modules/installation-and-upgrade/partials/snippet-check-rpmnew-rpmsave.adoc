. コンテナ内の[filename]`+*.rpmnew+`および[filename]`+*.rpmsave+`ファイルを確認してください：

+

[source,shell]
----
mgrctl exec 'find /etc/ -name "*.rpmnew" -o -name "*.rpmsave"'
----

+

[NOTE]
====
パッケージのインストール後に変更されたデフォルトの設定ファイルへの変更を含むアップグレードの場合、ファイルを上書きする代わりに、次のいずれかのファイルタイプが作成されます：

* [filename]`+*.rpmnew+`：新しいデフォルト設定を含み、変更されたファイルはそのまま残ります
* [filename]`+*.rpmsave+`：新しいデフォルトファイルによって置き換えられた、変更された設定ファイルのコピー
====

+

. [filename]`+*.rpmnew+`または[filename]`+*.rpmsave+`ファイルが見つかった場合は、その内容を確認し、望ましい変更をマージしてください。
