= コンテナの起動と停止
:revdate: 2024-02-13
:page-revdate: {revdate}

次のコマンドを使用して、{productname}{productnumber}サーバコンテナを再起動、起動、および停止できます。


サーバを`restart`{productname}{productnumber}起動するには、次のコマンドを実行してください。
----
# mgradm restart
5:23PM INF Welcome to mgradm
5:23PM INF Executing command: restart
----

サーバを`start`起動するには、次のコマンドを実行してください。
----
# mgradm start
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: start
----

サーバを`stop`起動するには、次のコマンドを実行してください。
----
# mgradm stop
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: stop
----
