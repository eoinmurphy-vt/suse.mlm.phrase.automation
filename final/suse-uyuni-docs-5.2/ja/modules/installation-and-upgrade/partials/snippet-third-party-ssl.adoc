.インポートした証明書の削除
[NOTE]
====
以前のバージョンでは、データベースコンテナには、次のサブジェクト代替名（SAN）が付与されたSSL証明書が必要でした:

* `db`
* `reportdb`

現在は不要です。
データベースコンテナには、外部に向けた完全修飾ドメイン名のみが必要です。
`db`および`reportdb`のSANを持つ古い証明書は引き続き使用できます。

同じ証明書をメインコンテナとデータベースコンテナの両方に使用できます。

新しい証明書をアップグレードコマンドに渡すには、`--ssl-db-ca-root`、`--ssl-db-cert`、および`--ssl-db-key`パラメータを使用します。
====
