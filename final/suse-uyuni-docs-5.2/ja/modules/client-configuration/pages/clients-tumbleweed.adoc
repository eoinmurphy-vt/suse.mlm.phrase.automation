このセクションでは、{opensuse}オペレーティングシステムを実行しているクライアントの登録について説明します。{productname}は{salt}を使用して{opensuse} Tumbleweedクライアントをサポートしています。

起動する機能は、{opensuse}クライアントの起動や、リポジトリの設定およびプロファイルの更新など、初期状態の実行をサポートします。



== ソフトウェアチャンネルの追加


{opensuse}クライアントを{productname}サーバに登録する前に、必要なソフトウェアチャネルを追加して同期する必要があります。

現在サポートされているアーキテクチャは、`x86_64`と`aarch64`です。
サポートされている製品とアーキテクチャの完全なリストについては、xref:client-configuration:supported-features.adoc[]をご覧ください。

== GPGキーの管理

クライアントは、ソフトウェアパッケージがインストールされる前に、その真正性を確認するためにGPGキーを使用します。
信頼できるソフトウェアのみがクライアントにインストールできます。

[IMPORTANT]
====
クライアントのセキュリティのために、GPGキーを信頼することは重要です。
どのキーが必要で信頼できるかを決定するのは管理者の仕事です。
GPG キーが信頼されていない場合、ソフトウェア チャンネルは使用できないため、クライアントにチャンネルを割り当てる決定はキーを信頼する決定に依存します。
====

GPG キーの詳細については、xref:client-configuration:gpg-keys.adoc[] を参照してください。
