[[products]]
= ソフトウェア製品
:revdate: 2024-05-10
:page-revdate: {revdate}

{productname}では、ソフトウェアが製品として提供されます。
あなたの{suse}サブスクリプションにより、さまざまな製品にアクセスでき、それらを{productname} {webui}で閲覧し、menu:Admin[セットアップウィザード> 製品]に移動して選択できます。

製品には、任意の数のソフトウェアチャネルが含まれています。
`Show product's channels`アイコンをクリックすると、製品に含まれるチャネルを確認できます。
製品を追加し、正常に同期が完了すると、製品が提供するチャネルにアクセスでき、{productname}サーバーおよびクライアントで製品内のパッケージを使用できます。
