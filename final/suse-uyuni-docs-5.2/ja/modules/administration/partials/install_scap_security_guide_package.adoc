修復を実行するには、Ansible制御ノードにSCAPセキュリティガイドパッケージをインストールする必要があります。

.プロシージャ:SCAPセキュリティガイドパッケージのインストール
[role=procedure]
_____
. menu:システム[概要]から、クライアントを選択します。
  次に、menu:ソフトウェア[パッケージ> インストール]をクリックします。

. `scap-security-guide`を検索し、お使いのシステムに適したパッケージをインストールしてください。
  パッケージの配布要件については、以下の表を参照してください：
+
[cols="1,1", options="header"]
.SCAPセキュリティガイドパッケージの要件
|===

| パッケージ名
| サポートされているシステム

| scap-security-guide
| {opensuse}, SLES12, SLES15

| scap-security-guide-redhat
| CentOS 7, CentOS 8, Fedora, Oracle Linux 7, Oracle Linux 8, RHEL7, RHEL8, RHEL9, Red Hat OpenStack Platform 10, Red Hat OpenStack Platform 13, Red Hat Virtualization 4, Scientific Linux

| scap-security-guide-debian
| Debian 12

| scap-security-guide-ubuntu
| Ubuntu 22.04

|===
_____

[NOTE]
====
Debian 13 および Ubuntu 24.04 の場合、{suse} はセキュリティガイド用のカスタムパッケージを提供していません。
それぞれのコミュニティリポジトリからアップストリームパッケージ（それぞれ`ssg-debian`および`ssg-debderived`）を使用する必要があります。
これらのアップストリームガイドは{suse}によって公式にサポートされていないことに注意してください。
====
